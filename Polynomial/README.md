Polynomial文件夹是源程序文件夹

使用方法：

	将这个文件夹中的所有程序都导入到QtCreator中，直接编译，运行即可。
	
	（注意，该方法可移植性较弱，可能需要对源代码进行简单的修改才能够编译通过）

包含：

	logindialog:

		logindialog.cpp 
		
		logindialog.h
		
		logindialog.ui
	
	widget:
		
		widget.cpp
		
		widget.h
		
		widget.ui
	
	others:
		
		main.cpp 
		
		connection.h 
		
		faction.h
